# Python Training

Daily Python exercises covering fundamentals to advanced concepts.

## Structure

- **Day 1** - Basics: Variables, data types, conditionals, simple operations
- **Day 2** - Intermediate: Loops, strings, lists, calculations  
- **Day 3** - Control Flow: Factorial, palindromes, number analysis
- **Day 4** - Data Structures: Dictionaries, sets, tuples, list operations
- **Day 5** - Object-oriented examples, classes, file handling, and project-style exercises

## Running Code

```bash
python filename.py
```

## Requirements

- Python 3.x

## Running Examples

Each file is self-contained and can be run independently. Most exercises prompt for user input, so be ready to provide values when running.

Example:
```
python day1/swap.py
```

## Learning Path

It is recommended to follow the exercises in order:

1. Start with Day 1 to understand Python basics
2. Progress to Day 2 to practice loops and data structures
3. Move to Day 3 for more complex problem-solving

## Notes

- Each file focuses on a specific concept or problem
- Most exercises are designed to be interactive
- Comments are included to explain the logic
- Experiment by modifying the code to deepen understanding

## Getting Started

Clone or download this repository and start with any file from day1 to begin learning Python.
